# Muster-Website Installateur

Professionelle Muster-Website für einen fiktiven Installateur-Betrieb
(**Müller Haustechnik**, Herrsching am Ammersee). Dient als Referenz auf
[aufregend.com](https://aufregend.com) für Handwerker, die sehen möchten,
wie ihre Website aussehen könnte.

## Live-Demo

Die Seite läuft via **GitHub Pages** auf:

👉 `https://<username>.github.io/muster-installateur/`

## Technik

- Reines **HTML / CSS / JavaScript** – keine Build-Tools, keine Abhängigkeiten
- **Responsive** für Handy, Tablet, Desktop
- **Google Fonts** (Inter + Plus Jakarta Sans)
- Bilder via **Unsplash** (frei nutzbar)

## Lokale Vorschau

Einfach die `index.html` im Browser öffnen.
